# Hogarth Frontend Dev Test

## Tech Used

- HTML5
- SCSS (Sass)
- No JavaScript
- No CSS frameworks

## Features

- Pixel-perfect implementation
- Responsive layouts at 1068px and 734px
- Responsive images using <picture>
- Clean, maintainable SCSS architecture

## Notes

- Line breaks and spacing matched to provided mockups
- Images used strictly from provided assets
- Media queries handled via Sass only

## How to Run

Open index.html in a browser or use Live Server.
